# VsLabyrinth 🎮

Un juego de laberinto dinámico desarrollado con JavaScript vanilla, donde los jugadores deben navegar a través de laberintos generados proceduralmente mientras compiten contra el tiempo.

## 📝 Descripción

VsLabyrinth es un juego web interactivo que genera laberintos únicos con diferentes niveles de dificultad. Los jugadores controlan un personaje que debe encontrar la salida mientras se registra su tiempo.

## 🚀 Características

- Generación procedural de laberintos
- Tres niveles de dificultad: Fácil, Medio y Difícil
- Sistema de cronometraje
- Controles responsivos (teclas WASD y flechas)
- Diseño responsive
- Efectos visuales dinámicos
- Interfaz intuitiva

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Canvas API

## 🎮 Cómo Jugar

1. Selecciona un nivel de dificultad (Fácil, Medio o Difícil)
2. Usa las teclas de flecha o WASD para mover al jugador
3. Encuentra la salida (casilla verde)
4. Intenta completar el laberinto en el menor tiempo posible

## ⌨️ Controles

- **W** o **↑**: Mover arriba
- **S** o **↓**: Mover abajo
- **A** o **←**: Mover izquierda
- **D** o **→**: Mover derecha

## 🎯 Niveles de Dificultad

- **Fácil**: Laberinto 15x15
- **Medio**: Laberinto 21x21
- **Difícil**: Laberinto 25x25
